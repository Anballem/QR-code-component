# QR Code Component

A simple responsive QR code card component built with **HTML** and **CSS** (Frontend Mentor challenge).

## Live preview

Open `index.html` in your browser.

## How to run locally

1. Make sure you have the project files on disk.
2. Open **`index.html`**.

There are no build tools or dependencies required.

## Project structure

- `index.html` — page markup
- `style.css` — styling for layout/typography/card
- `images/` — static assets (e.g. `image-qr-code.png`)
